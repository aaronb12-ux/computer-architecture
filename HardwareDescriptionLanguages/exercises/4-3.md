# Four-Input XOR HDL Module

Write an HDL module that computes a four-input XOR function. The input is `a[3:0]`, and the output is `y`.

```systemverilog
module fourInputXOR(input logic [3:0] a,
                    output logic y);

    assign y = a[0] ^ a[1] ^ a[2] ^ a[3];

endmodule
```
