# Exercise 1 — Simplified Circuit Schematic

## Problem

Sketch a schematic of the circuit described by the following HDL code and simplify it so the schematic uses the minimum number of gates.

```systemverilog
module exercise1(
    input  logic a, b, c,
    output logic y, z
);

    assign y = a & b & c | a & b & ~c | a & ~b & c;

    assign z = a & b | ~a & ~b;

endmodule
```

---

# Inputs and Outputs

## Inputs
- `a`
- `b`
- `c`

## Outputs
- `y`
- `z`

---

# Boolean Simplification

## Output `y`

Original expression:

```text
y = a & b & c | a & b & ~c | a & ~b & c
```

Factor out `a`:

```text
y = a(bc + b~c + ~bc)
```

Simplify:

```text
bc + b~c = b
```

So:

```text
y = a(b + ~bc)
```

Apply absorption:

```text
b + ~bc = b + c
```

Final simplified expression:

```text
y = a(b + c)
```

---

## Output `z`

Original expression:

```text
z = a & b | ~a & ~b
```

This is the XNOR function:

```text
z = a XNOR b
```

---

# Simplified Schematic

## Logic for `y`

```text
y = a(b + c)
```

Implementation:
1. OR gate for `b + c`
2. AND gate with input `a`

---

## Logic for `z`

```text
z = a XNOR b
```

Implementation:
1. Single XNOR gate
