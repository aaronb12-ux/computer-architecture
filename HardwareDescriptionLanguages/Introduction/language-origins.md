# Language Origins

**Verilog** was developed by Gateway Design Automation in 1984 as a proprietary language for logic simulation.

- 1989: Gateway was acquired by Cadence  
- 1990: Verilog became an open standard under Open Verilog International (OVI)  
- 1995: Verilog became an IEEE standard  

In 2005, the language was extended to:
- Fix inconsistencies and quirks  
- Improve support for system modeling and verification  

These extensions were unified into a single standard called **SystemVerilog**.

- SystemVerilog file extension: `.sv`

---

## Verilog vs SystemVerilog

- Both languages can describe **any hardware system**  
- Both have their own quirks and design styles  

### Choosing a Language
The best choice is usually:
- The one already used in your organization  
- The one required by customers or tools  

---

## Interoperability

Modern CAD tools allow:
- Mixing Verilog and SystemVerilog  
- Different modules to be written in different languages  

This makes it possible to integrate legacy designs with newer system-level features.
