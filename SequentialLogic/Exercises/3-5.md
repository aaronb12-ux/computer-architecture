## D Flip-Flop Timing Diagram Problem

**Given the below CLK and D waveforms in the image, sketch the output Q:**

![CamScanner 4-2-26 22 37n](https://github.com/user-attachments/assets/97926dff-87a4-46ec-ab8d-4381aa6dd7c9)

### Solution:

For a D flip-flop, D only gets copied to Q during the rising clock edge. So for example, if the clock is currently at 0, and D = 1 and Q = 1, if D falls to 0, Q does not change because the clock has not been triggered.

After this happens, and there is a rising clock edge, then Q will become 0 because that's the current value in D. There is no copy from D to Q during a falling clock edge (1 → 0).

**Key Points:**
- **Rising edge (0 → 1):** Q copies the current value of D
- **Falling edge (1 → 0):** Q holds its previous value (no change)
- **Between clock edges:** Q remains constant regardless of D changes

**Procedure:**
1. Identify each rising clock edge
2. At each rising edge, sample the current value of D
3. Set Q to that sampled value immediately after the rising edge
4. Q holds this value until the next rising edge
