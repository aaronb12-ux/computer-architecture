## T Flip-Flop Design Problem

**Problem:** A toggle (T) flip-flop has one input, CLK, and one output, Q. On each rising edge of CLK, Q toggles to the complement of its previous value. Draw a schematic for a T flip-flop using a D flip-flop and an inverter.

**Explanation:**

A T flip-flop toggles its previous value on the rising clock edge.

**How it works:**
1. Connect the output Q back through an inverter to the input D
2. On each rising clock edge, the D flip-flop samples D (which is Q̄, the complement of Q)
3. This causes Q to become Q̄ on each rising edge, effectively toggling

**Behavior:**
- If Q = 0, then D = Q̄ = 1, so on the next rising edge Q becomes 1
- If Q = 1, then D = Q̄ = 0, so on the next rising edge Q becomes 0
- On falling clock edges and between edges, the output stays the same (standard D flip-flop behavior)
