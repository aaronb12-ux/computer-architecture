### Overview

In a **Mealy Model Machine**, both the **input and output are labeled on the transition arrows**.

---

### Finding the Next State Equations

The process for finding next state equations in a Mealy machine is very similar to that of a Moore machine:

1. **Encode the states and outputs**
2. **Create the state transition table**

The transition table should have the following columns:
- Current State  
- Input  
- Next State  

---

### Handling Inputs

Some inputs may be expressed as Boolean equations (e.g., `AB` or `A' + B'`).

- When constructing the table, **focus only on the input combinations**, not the logic expression itself.
- For example, if the input is `A + B`, you must include all input combinations where the expression is true:
  - `1 1`
  - `1 0`
  - `0 1`

---

### Deriving Next State Equations

- Use the **Sum of Products (SOP)** method to derive:
  - \( S'_1 \)
  - \( S'_0 \)

---

### Output Table

- For a **Moore machine**, the output table has:
  - State  
  - Output  

- For a **Mealy machine**, the output table includes:
  - State  
  - Input  
  - Output  

---
