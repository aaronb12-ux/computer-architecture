### Solution:

Whenever there is a rise in the input waveforms of S and R, that signifies a 1, and a fall is a 0. We follow the SR latch truth table to determine when the output should rise (1) or fall (0).

**SR Latch Truth Table:**
- S = 1, R = 0 → Q = 1 (Set)
- S = 0, R = 1 → Q = 0 (Reset)
- S = 0, R = 0 → Q = previous value (Hold)
- S = 1, R = 1 → Q = invalid state (Not allowed)
